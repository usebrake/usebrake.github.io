Install guide

Install Brake for Windows

Brake now installs with a normal Windows installer. The setup file includes everything normal users need.

Download BrakeSetup.exe View source on GitHub
Windows may show a SmartScreen warning while Brake is in beta and unsigned. Only download Brake from the official GitHub release or this website.
1

Download the installer

Click the download button and save BrakeSetup.exe from the latest GitHub release.

Screenshot placeholderAdd a screenshot of the browser download prompt or GitHub release asset here.
2

Run BrakeSetup.exe

Double-click the installer. If Windows asks whether you want to allow changes, choose Yes.

Screenshot placeholderAdd a screenshot of the installer file or Windows admin prompt here.
3

Finish setup

Follow the installer window. Brake copies itself to C:\Program Files\Brake, registers its background services, and creates shortcuts.

Screenshot placeholderAdd a screenshot of the installer completion screen here.
4

Open Brake

Launch Brake from the installer, desktop shortcut, or Start Menu. Save your recovery code on first launch, then turn on protection.

Screenshot placeholderAdd a screenshot of the first Brake screen or recovery-code step here.

Updating

Download the newest BrakeSetup.exe and run it again. It replaces the app files while keeping your local settings in C:\ProgramData\Brake.

Source code

The source is public for review. Normal users should install with the EXE; developers can still build from source.

Uninstalling

Use Windows Settings > Apps > Installed apps > Brake > Uninstall, or run the uninstaller from C:\Program Files\Brake.

Install problems

If setup fails, restart Windows and run the installer again. If it still fails, report the issue on GitHub.
