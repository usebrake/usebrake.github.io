Install the free beta

Brake installs background services, so Windows will ask for administrator permission during setup.
1
Step 1

Download the installer

Download BrakeSetup.exe from the latest GitHub release or click the yellow button below to download directly.

Download BrakeSetup.exe
2
Step 2

Run BrakeSetup.exe

Double-click the installer. Windows may warn that the file is from an unknown publisher because the beta is not code-signed yet. Choose to run it anyway, then approve the Windows admin prompt. Brake installs its app files, shortcuts, and background services.

Installs to C:\Program Files\Brake
3
Step 3

Open Brake

Launch Brake from the installer, desktop shortcut, or Start Menu. On first launch, save your recovery code before turning protection on.

Ready to turn on protection

First launch

On first launch, Brake shows a recovery code. Write it down, take a photo, or give it to someone you trust. Do not save it somewhere easy to reach on the same computer if you want stronger commitment.

What gets installed

Brake copies the app to C:\Program Files\Brake, creates shortcuts, and registers background services. After install, you can delete the downloaded setup file.

Need help?

If install fails, open a GitHub issue and say which step failed. Do not include passwords, recovery codes, or explicit screenshots.
